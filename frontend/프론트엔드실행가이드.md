## 🚀 실행 방법

### 1단계: 패키지 설치

```bash
cd frontend
npm install
```

TailwindCSS 추가 패키지:
```bash
npm install -D tailwindcss postcss autoprefixer
```

### 2단계: 프론트엔드 서버 실행

```bash
npm run dev
```

서버가 다음 주소에서 실행됩니다:
```
http://localhost:3001
```

## 🔗 API 연동

프론트엔드는 Vite 프록시를 통해 백엔드 API와 연동됩니다:

- 프론트엔드: `http://localhost:3001`
- 백엔드: `http://localhost:3000`
- API 요청: `/api/*` → `http://localhost:3000/api/*`

## ⚠️ 주의사항

### 백엔드 서버 실행 필수
프론트엔드를 실행하기 전에 백엔드 서버가 실행되어 있어야 합니다:

```bash
cd backend
npm run dev
```

### CORS 설정
백엔드에서 CORS가 허용되어 있어야 합니다. 현재 설정:
- Origin: `http://localhost:3001`
- Credentials: `true`

### 환경 변수
현재는 `.env` 파일을 사용하지 않지만, 필요시 추가할 수 있습니다:

```env
VITE_API_URL=http://localhost:3000
```

## 🐛 문제 해결

### 1. 포트 충돌
다른 프로그램이 포트 3001을 사용 중이면:

```javascript
// vite.config.js
server: {
  port: 3002, // 포트 변경
}
```

### 2. API 연결 실패
- 백엔드 서버가 실행 중인지 확인
- 브라우저 콘솔에서 네트워크 탭 확인
- CORS 에러가 있으면 백엔드 CORS 설정 확인

### 3. TailwindCSS 스타일 적용 안 됨
```bash
# TailwindCSS 재설치
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
