## 기술 스택

### 프론트엔드
- **프레임워크:** React 18 + TypeScript
- **빌드 도구:** Vite 5
- **스타일링:** Tailwind CSS 3
- **라우팅:** React Router DOM 6
- **HTTP 클라이언트:** Axios (인터셉터로 JWT 자동 첨부)
- **상태 관리:** Zustand
- **차트:** Recharts
- **포트:** 3001 (개발), Nginx 프록시 (배포)

### 백엔드
- **프레임워크:** Fastify 4
- **ORM:** Prisma 5 + PostgreSQL
- **인증:** JWT (jsonwebtoken), bcrypt
- **이메일:** Nodemailer (Gmail SMTP)
- **보안:** @fastify/helmet, @fastify/rate-limit@9, @fastify/cors
- **검증:** Zod
- **포트:** 3000

### 인프라
- **서버:** AWS Lightsail (Ubuntu, IP: 43.203.197.122)
- **웹서버:** Nginx (리버스 프록시)
- **프로세스 관리:** PM2
- **데이터베이스:** PostgreSQL
- **SSH:** bitnami 사용자, PEM 키 인증

## 배포 절차

### 프론트엔드
```bash
cd frontend
rm -rf dist && npx vite build
scp -i "LightsailDefaultKey-ap-northeast-2.pem" -r dist/* bitnami@43.203.197.122:/home/bitnami/contract-m/frontend/dist/
```

### 백엔드
```bash
scp -i "LightsailDefaultKey-ap-northeast-2.pem" -r backend/src/* bitnami@43.203.197.122:/home/bitnami/contract-m/backend/src/
ssh -i "..." bitnami@43.203.197.122 "npx pm2 restart all"
```

### 데이터베이스 마이그레이션
```bash
ssh -i "..." bitnami@43.203.197.122 "cd /home/bitnami/contract-m/backend && npx prisma migrate deploy"
```
