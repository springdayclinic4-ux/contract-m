### Phase 1: 사용자 계정 시스템 만들기 (약 2주)

#### 1-1. 회원가입 기능
**무엇을 하나요?**
- 병원에서 이 웹사이트를 사용하려면 계정이 필요합니다
- 회원가입 페이지를 만들어서 병원 정보를 입력받습니다

**어떻게 작동하나요?**
1. 병원명, 사업자등록번호, 이메일, 비밀번호 등을 입력
2. 이메일로 인증 코드를 보냄
3. 인증 코드를 입력하면 회원가입 완료

**왜 필요한가요?**
- 각 병원의 정보를 저장해서 나중에 다시 사용할 수 있게 하기 위해
- 계약서를 저장하고 관리하기 위해

#### 1-2. 로그인 기능
**무엇을 하나요?**
- 회원가입한 병원이 다시 접속할 때 사용하는 기능

**어떻게 작동하나요?**
1. 이메일과 비밀번호 입력
2. 맞으면 로그인 성공, 틀리면 오류 메시지

**왜 필요한가요?**
- 매번 병원 정보를 입력하는 번거로움을 줄이기 위해

## 🛠️ 개발 과정 설명 (비개발자용)

### 개발이란?
**간단히 말하면:** 컴퓨터가 이해할 수 있는 언어로 프로그램을 만드는 것입니다.

**예를 들어:**
- 웹사이트를 만드는 것
- 앱을 만드는 것
- 자동화 시스템을 만드는 것

### 우리 프로젝트는 어떻게 만들어지나요?

#### 1. 프론트엔드 (사용자가 보는 화면)
**무엇인가요?**
- 사용자가 브라우저에서 보는 화면과 버튼들
- 지금 보시는 계약서 작성 화면이 프론트엔드입니다

**무엇으로 만드나요?**
- HTML: 화면 구조
- CSS: 디자인 (색상, 크기, 위치 등)
- JavaScript: 버튼 클릭, 계산 등 동작

**현재 상태:** ✅ 기본 화면은 완성됨

#### 2. 백엔드 (서버, 데이터 저장소)
**무엇인가요?**
- 사용자가 보지 못하는 뒷단
- 데이터를 저장하고, 로그인을 확인하고, 계산을 처리하는 부분

**무엇으로 만드나요?**
- Node.js 또는 Python: 서버 프로그램
- 데이터베이스: 정보를 저장하는 곳 (책장 같은 것)

**현재 상태:** ❌ 아직 만들어지지 않음 (다음 단계)

#### 3. 데이터베이스 (정보 저장소)
**무엇인가요?**
- 모든 정보를 저장하는 곳
- 예: 병원 정보, 계약서 내용, 사용자 계정 등

**무엇으로 만드나요?**
- PostgreSQL 또는 MySQL: 데이터베이스 프로그램

**현재 상태:** ❌ 아직 만들어지지 않음 (다음 단계)

## 💡 비개발자를 위한 용어 설명

### API (Application Programming Interface)
**쉽게 설명:** 다른 프로그램과 정보를 주고받는 창구
**예시:** 공공데이터에서 병원 정보를 가져올 때 사용

### 데이터베이스
**쉽게 설명:** 정보를 체계적으로 저장하는 전자 파일장
**예시:** 엑셀 시트처럼 정리된 정보 저장소

### 서버
**쉽게 설명:** 24시간 돌아가는 컴퓨터 (클라우드에 있음)
**예시:** 웹사이트가 항상 접속 가능하도록 하는 컴퓨터

### 프론트엔드 vs 백엔드
**프론트엔드:** 사용자가 보는 화면 (식당의 홀)
**백엔드:** 사용자가 보지 못하는 뒷단 (식당의 주방)

### 배포
**쉽게 설명:** 만든 프로그램을 실제로 사용할 수 있게 공개하는 것
**예시:** 웹사이트를 인터넷에 올려서 누구나 접속할 수 있게 함

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 개발이 얼마나 걸리나요?
**A:** 전체 기능 완성까지 약 10주 (2.5개월) 예상됩니다.

### Q2. 지금 사용할 수 있나요?
**A:** 네, 기본 계약서 생성 기능은 지금 바로 사용 가능합니다. 다만 계약서를 저장하거나 통계를 보는 기능은 아직 없습니다.

### Q3. 개발자가 필요하나요?
**A:** 네, 다음 단계부터는 백엔드 개발자가 필요합니다. 프론트엔드 개발자도 함께 있으면 더 빠르게 진행됩니다.

### Q4. 비용이 얼마나 드나요?
**A:** 
- 개발 인력: 개발자 1명 기준 약 10주
- 서버 호스팅: 월 약 5-10만원
- 도메인: 연간 약 1-2만원
- 외부 API: 대부분 무료 (공공데이터)

### Q5. 완성되면 어떻게 사용하나요?
**A:** 
1. 병원에서 회원가입
2. 로그인
3. 계약서 작성 (지금처럼)
4. 저장 버튼 클릭
5. 나중에 다시 불러와서 수정하거나 PDF 다운로드
6. 통계 화면에서 전체 현황 확인
