## ⚠️ 주의사항

### AWS SDK v2 경고
서버 실행 시 AWS SDK v2가 종료 지원(end-of-support)되었다는 경고가 표시됩니다.
이것은 기능에 영향을 주지 않지만, 나중에 AWS SDK v3로 업그레이드하는 것이 좋습니다.

```
The AWS SDK for JavaScript (v2) has reached end-of-support.
Please migrate your code to use AWS SDK for JavaScript (v3).
```

### 해결 방법 (선택사항)
```bash
npm uninstall aws-sdk
npm install @aws-sdk/client-ses
```

그 후 `src/utils/email.js`를 AWS SDK v3 문법으로 수정

## 🎯 성공적으로 해결한 문제들

1. **Prisma 7.x 엔진 타입 문제**
   - Prisma 7.2.0 → 5.19.1로 다운그레이드
   - `prisma.config.ts` 제거, `schema.prisma`에 `url` 추가

2. **PostgreSQL 인증 오류**
   - `.env` 파일 생성 및 `DATABASE_URL` 설정
   - 비밀번호 URL 인코딩 처리

3. **Zod 스키마 검증 문제**
   - Fastify와 Zod 호환성 문제 해결
   - `zodValidation` 미들웨어 생성
   - `schema:` → `preValidation:` 변경

4. **Node.js 24 호환성**
   - `@fastify/jwt` → `jsonwebtoken` 변경
   - `.npmrc`에 `engine-strict=false` 추가
