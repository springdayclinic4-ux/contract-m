## 다음 단계 (순서대로 실행)

### 1단계: 데이터베이스 생성

**옵션 A: 자동 스크립트**
```cmd
데이터베이스생성.bat
```

**옵션 B: pgAdmin 사용**
- pgAdmin 실행
- localhost 서버 연결 (postgres / password123)
- 데이터베이스 우클릭 → Create → Database
- 이름: `hos_contracts`

**옵션 C: Prisma Migrate로 자동 생성** (데이터베이스가 없어도 시도 가능)
```cmd
npm run db:migrate
```

### 2단계: Prisma 클라이언트 생성

```cmd
npm run db:generate
```

### 3단계: 데이터베이스 마이그레이션 (1단계에서 안 했다면)

```cmd
npm run db:migrate
```

마이그레이션 이름: `init`

### 4단계: 서버 실행

```cmd
npm run dev
```

정상 실행되면:
- 콘솔에 "🚀 Server is running on port 3000" 표시
- http://localhost:3000/health 접속하여 확인

---

## 문제 해결

### Prisma generate 오류
- `prisma/config.ts` 파일이 있는지 확인
- `.env` 파일의 `DATABASE_URL` 확인

### 데이터베이스 연결 오류
- PostgreSQL이 실행 중인지 확인
- 비밀번호가 `password123`인지 확인
- 데이터베이스 `hos_contracts`가 생성되었는지 확인

### 포트 사용 중 오류
- 다른 프로세스가 3000 포트를 사용 중일 수 있음
- `.env`에서 `PORT=3001` 등으로 변경 가능
