# PostgreSQL 데이터베이스 설정 가이드 (Windows)

## 방법 1: PostgreSQL 설치 경로에서 실행

PostgreSQL이 설치되어 있다면, 설치 경로에서 직접 실행할 수 있습니다:

### 일반적인 설치 경로:
```
C:\Program Files\PostgreSQL\18\bin\psql.exe
```

### 실행 방법:

**명령 프롬프트에서:**
```cmd
"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres
```

또는 PostgreSQL 버전이 다르면:
```cmd
"C:\Program Files\PostgreSQL\17\bin\psql.exe" -U postgres
"C:\Program Files\PostgreSQL\16\bin\psql.exe" -U postgres
```

### 데이터베이스 생성:

PostgreSQL 접속 후:
```sql
CREATE DATABASE hos_contracts;
\q
```

---

## 방법 2: pgAdmin 사용 (GUI)

1. **pgAdmin 실행** (PostgreSQL 설치 시 함께 설치됨)
2. 서버 연결 (localhost, postgres 사용자)
3. 데이터베이스 우클릭 → Create → Database
4. 이름: `hos_contracts` 입력
5. Save

---

## 방법 3: Prisma Migrate로 자동 생성

Prisma Migrate가 데이터베이스를 자동으로 생성할 수도 있습니다:

```cmd
npm run db:migrate
```

마이그레이션 이름: `init`

---

## 방법 4: PATH에 추가 (영구적 해결)

PostgreSQL bin 폴더를 시스템 PATH에 추가:

1. **시스템 속성** → **고급** → **환경 변수**
2. **시스템 변수**에서 `Path` 선택 → **편집**
3. **새로 만들기** → PostgreSQL bin 경로 추가:
   ```
   C:\Program Files\PostgreSQL\18\bin
   ```
4. 확인 후 새 터미널에서 `psql` 명령어 사용 가능
