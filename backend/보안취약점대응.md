# 보안 취약점 대응 가이드

## 현재 상황

npm audit 결과 6개의 취약점이 발견되었습니다:
- 1개 low
- 5개 high

## 취약점 분석

### 1. aws-sdk (High)
- **문제**: AWS SDK v2 사용 중
- **해결**: v3로 마이그레이션 권장 (breaking changes 있음)
- **우선순위**: 중간 (이메일 발송 기능 사용 시)

### 2. hono (High) - Prisma 의존성
- **문제**: Prisma의 개발 도구 의존성
- **영향**: 개발 환경에서만 사용, 프로덕션에는 영향 없음
- **우선순위**: 낮음

### 3. tar (High) - 의존성 패키지
- **문제**: 빌드 도구의 의존성
- **영향**: 개발 환경에서만 사용
- **우선순위**: 낮음

## 권장 조치

### 즉시 조치 불필요
현재 발견된 취약점들은:
- 대부분 개발 도구의 의존성
- 프로덕션 환경에 직접적인 영향 없음
- 직접 사용하지 않는 패키지들

### 향후 조치 (선택사항)

#### 1. AWS SDK v3 마이그레이션 (나중에)
```bash
# aws-sdk v2 제거
npm uninstall aws-sdk

# AWS SDK v3 설치 (클라이언트별로 설치)
npm install @aws-sdk/client-ses
```

그 후 `backend/src/utils/email.js` 파일을 v3 API로 수정 필요.

#### 2. Prisma 업데이트 (Prisma 팀이 해결할 때까지 대기)
Prisma가 hono 의존성을 업데이트하면 자동으로 해결됩니다.
