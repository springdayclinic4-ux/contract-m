# ✅ AWS 배포 체크리스트

배포 전후 확인사항을 체크하세요!

---

## 📋 배포 전 체크리스트

### 코드 준비
- [ ] 모든 변경사항이 Git에 커밋됨
- [ ] GitHub에 코드 푸시 완료
- [ ] `.gitignore`에 `.env` 파일 포함 확인
- [ ] `package.json`의 의존성 버전 확인
- [ ] 프론트엔드 로컬 빌드 테스트 (`npm run build`)
- [ ] 백엔드 로컬 테스트 완료

### 환경 변수 준비
- [ ] `JWT_SECRET` 생성 (최소 32자)
- [ ] `JWT_REFRESH_SECRET` 생성 (최소 32자)
- [ ] 데이터베이스 연결 문자열 확인
- [ ] CORS 설정 확인
- [ ] AWS 리전 선택 완료

### AWS 계정
- [ ] AWS 계정 생성 완료
- [ ] 신용카드 등록 완료
- [ ] IAM 사용자 생성 (선택)
- [ ] AWS CLI 설치 (선택)

---

## 🗄️ 데이터베이스 체크리스트

### RDS / Neon
- [ ] PostgreSQL 16 인스턴스 생성
- [ ] 마스터 사용자 이름/비밀번호 저장
- [ ] 엔드포인트 복사
- [ ] 보안 그룹 / 방화벽 규칙 설정
- [ ] 연결 테스트 완료
- [ ] 연결 문자열 생성

```
postgresql://user:password@endpoint:5432/dbname
```

## 🔧 백엔드 체크리스트

### Elastic Beanstalk
- [ ] 애플리케이션 생성
- [ ] 환경 생성 (Node.js 플랫폼)
- [ ] ZIP 파일 업로드
- [ ] 환경 변수 설정:
  - [ ] `NODE_ENV=production`
  - [ ] `PORT=3001`
  - [ ] `DATABASE_URL`
  - [ ] `JWT_SECRET`
  - [ ] `JWT_REFRESH_SECRET`
  - [ ] `ALLOWED_ORIGINS`
  - [ ] `MOCK_EMAIL=true`
- [ ] 배포 완료 확인
- [ ] 헬스 체크 통과 확인
- [ ] 백엔드 URL 복사

### Lightsail
- [ ] 인스턴스 생성 (Node.js 블루프린트)
- [ ] SSH 연결 테스트
- [ ] Git 저장소 클론
- [ ] 백엔드 의존성 설치
- [ ] `.env` 파일 생성
- [ ] PM2로 백엔드 실행
- [ ] 방화벽 규칙 설정 (포트 3001)
- [ ] 고정 IP 할당

### EC2
- [ ] 인스턴스 생성 (Ubuntu 22.04)
- [ ] 키 페어 다운로드
- [ ] 보안 그룹 설정
- [ ] Elastic IP 할당
- [ ] SSH 연결 테스트
- [ ] `setup-ec2.sh` 스크립트 실행
- [ ] `.env` 파일 편집
- [ ] PM2 상태 확인
