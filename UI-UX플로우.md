# 의사 일용직 근로계약서 생성 및 전자서명 시스템 - UI/UX 플로우

## 🏗️ 플랫폼 아키텍처 개요

**참고**: 상세한 HOS Data Structure는 `HOS_Data_Structure.md` 파일을 참고하세요.

본 시스템은 **HOS (Hospital Operating System) 플랫폼**의 일부로 설계되며, 향후 확장 가능한 구조를 가지고 있습니다. **HOS 구조가 메인 아키텍처**이며, 모든 기능은 이 구조를 기반으로 설계됩니다.

### 시스템 구조
```
Platform
├── 사용자 Domain (User Domain)
│   └── 개인정보 본인인증, 계정 인증
│
├── HOS Domain (Hospital Operating System) ← 현재 시스템
│   ├── 조직 DB (Organization 데이터)
│   │   ├── Organization Definition (조직 정의)
│   │   ├── Member Information (멤버 소속 정보)
│   │   ├── Role, Team, Permission (권한 관리)
│   │   └── Member_Team_Role Assignment
│   │
│   └── 계약서 관리 모듈 (현재 개발 중)
│       ├── 의사 일용직 근로계약서
│       └── 일반 근로계약서
```

### 시스템 흐름
1. **사용자 회원가입** (개인정보 본인인증) → 사용자 Domain
2. **사용자 로그인** (계정 인증) → 사용자 Domain
3. **HOS 접속 후 시스템 사용** → HOS Domain
   - 소속 조직 확인
   - Session 발급
   - 조직별 권한 확인
   - 계약서 관리 모듈 접근

**참고**: 현재 단계에서는 계약서 관리 모듈만 구현하지만, 향후 HOS 플랫폼 확장 시 조직 DB와 완전 통합될 예정입니다.

```
┌─────────────────────────────────────┐

│  │                               │  │
│  │  비밀번호 *                   │  │
│  │  [___________________] [👁]    │  │
│  │                               │  │
│  │  [ ] 로그인 상태 유지          │  │
│  │                               │  │

│  │                               │  │

│                                     │

```

#### 주요 요소
- **이메일 입력 필드**: 
  - 이메일 형식 실시간 검증
  - 오류 시 빨간색 테두리 및 메시지 표시
- **비밀번호 입력 필드**: 
  - 비밀번호 표시/숨김 토글 (👁 아이콘)
  - 오류 시 빨간색 테두리 및 메시지 표시
- **로그인 상태 유지 체크박스**: 
  - 체크 시 7일간 자동 로그인 유지
- **로그인 버튼**: 
  - 클릭 시 로딩 상태 표시 (버튼 비활성화, "로그인 중..." 표시)
  - 성공/실패 피드백 제공
- **회원가입 링크**: 
  - 파란색 링크, 회원가입 페이지로 이동
- **비밀번호 찾기 링크**: 
  - 파란색 링크, 비밀번호 찾기 페이지로 이동

#### 사용자 행동
1. 이메일 입력 (자동 포커스)
2. 비밀번호 입력
3. (선택) 로그인 상태 유지 체크
4. 로그인 버튼 클릭
5. **성공 시**:

#### 입력 필드 상세 설명

**병원명**

**사업자등록번호**
- 필수 입력
- 형식: 10자리 숫자 (예: 1234567890)
- 실시간 형식 검증
- 중복 체크 (이미 가입된 사업자등록번호인지 확인)
- **중요**: 사업자등록번호는 각 병원의 고유 ID로 사용됩니다
- **자동 연동 버튼**: 클릭 시 공공데이터에서 병원 정보 자동 불러오기
  - 병원명, 대표자명, 주소 자동 입력
  - 불러온 정보는 수정 가능

**대표자 성명**
- 필수 입력
- 최대 20자

**병원 주소**
- 필수 입력
- 주소 검색 버튼: 도로명주소 API 연동
- 상세주소 입력 필드 추가

**병원 연락처**
- 필수 입력
- 형식: 02-1234-5678 또는 010-1234-5678
- 자동 하이픈(-) 삽입
- 병원 대표 전화번호

**담당자명**
- 선택 입력
- 최대 20자
- 담당자가 있는 경우에만 입력

**담당자 연락처**
- 선택 입력
- 형식: 010-1234-5678
- 담당자명을 입력한 경우 필수 입력으로 변경
- 자동 하이픈(-) 삽입

**이메일**
- 필수 입력
- 이메일 형식 검증
- 중복 체크 (이미 가입된 이메일인지 확인)
- 안내 문구: "이메일은 로그인 ID로 사용됩니다"

**비밀번호**
- 필수 입력
- 조건: 8자 이상, 영문+숫자+특수문자 조합
- 실시간 강도 표시 (약함/보통/강함)
- 표시/숨김 토글

**비밀번호 확인**
- 필수 입력
- 실시간 일치 여부 확인
- 일치 시: ✓ "비밀번호가 일치합니다" (초록색)
- 불일치 시: ⚠️ "비밀번호가 일치하지 않습니다" (빨간색)

**약관 동의**
- 전체 동의 체크박스: 모든 약관 일괄 동의/해제
- 서비스 이용약관: 필수, 약관 보기 링크
- 개인정보 처리방침: 필수, 약관 보기 링크
  - 회사가 제공하는 다른 서비스 정보 공유 내용 포함
- 개인정보 제3자 제공 동의: 필수, 약관 보기 링크
  - 회사가 제공하는 다른 서비스 정보 공유 내용 포함
- 마케팅 정보 수신 동의: 선택, 약관 보기 링크

## 🎯 주요 사용자 시나리오

### 시나리오 1: 처음 방문 및 회원가입

1. **웹사이트 접속** → 자동으로 로그인 페이지로 이동
2. **"회원가입"** 링크 클릭
3. **기본 정보 입력**:
   - 병원명 입력
   - 사업자등록번호 입력 → **"자동 연동"** 버튼 클릭 (병원 정보 자동 입력)
   - 대표자 성명 확인/수정
   - 병원 주소 확인/수정
   - 연락처 입력
   - 이메일 입력 (중복 체크)
   - 비밀번호 입력 (강도 표시 확인)
   - 비밀번호 확인 (일치 여부 확인)
   - 약관 동의 (전체 동의 또는 개별 동의)
4. **"다음 단계"** 버튼 클릭
5. **이메일 인증**:
   - 입력한 이메일로 인증 코드 발송 확인
   - 인증 코드 입력 (6자리)
   - "인증하기" 버튼 클릭
6. **회원가입 완료**:
   - 자동 로그인 처리
   - 대시보드로 자동 이동

### 시나리오 2: 로그인 및 첫 계약서 작성

1. **로그인 페이지 접속** (처음 방문 시 자동 이동)
2. **이메일과 비밀번호 입력**
3. **"로그인"** 버튼 클릭 → 통합 대시보드 접속
4. **"일용직 근로계약서 작성기"** 서비스 카드 클릭
5. **일용직 근로계약서 작성기 대시보드** 접속
6. **"계약서 생성하기"** 버튼 클릭
7. **병원 정보**: 자동으로 로드됨 (수정 가능)
8. **의사 정보**: 
   - 저장된 의사가 있으면 선택
   - 없으면 직접 입력
9. **근무 조건**: 날짜, 시간 입력
10. **급여 정보**: 세전 또는 세후 금액 입력 (자동 계산)
11. **미리보기 확인**: 오른쪽에서 실시간 확인
12. **저장**: 서버에 저장
13. **PDF 다운로드**: 인쇄 또는 보관

### 시나리오 3: 비밀번호 찾기

1. **로그인 페이지** 접속
2. **"비밀번호 찾기"** 링크 클릭
3. **이메일 입력**: 가입 시 사용한 이메일 입력
4. **"인증 코드 발송"** 버튼 클릭
5. **이메일 확인**: 받은 인증 코드 확인
6. **인증 코드 입력**: 6자리 코드 입력
7. **"인증하기"** 버튼 클릭
8. **비밀번호 재설정**:
   - 새 비밀번호 입력
   - 새 비밀번호 확인
   - "비밀번호 재설정" 버튼 클릭
9. **재설정 완료**: 로그인 페이지로 이동
10. **로그인**: 새 비밀번호로 로그인

### 시나리오 4: 기존 계약서 수정

1. **통합 대시보드** 접속
2. **"일용직 근로계약서 작성기"** 선택
3. **일용직 근로계약서 작성기 대시보드** 접속
4. **"계약서"** 메뉴 클릭 → 계약서 목록 접속
5. **검색/필터**: 원하는 계약서 찾기
6. **"보기"** 클릭 → 상세 페이지
7. **"수정"** 버튼 클릭
8. **정보 수정**
9. **저장**: 변경사항 저장
10. **PDF 재다운로드**: 필요시

### 시나리오 5: 통계 확인

1. **통합 대시보드** 접속
2. **"일용직 근로계약서 작성기"** 선택
3. **일용직 근로계약서 작성기 대시보드** 접속
4. **"통계"** 메뉴 클릭
5. **기간 선택**: 예) 2024년 1월
6. **필터 적용**: 특정 의사만 보기 (선택)
7. **차트 확인**: 월별 추이, 의사별 통계 등
8. **엑셀 다운로드**: 회계팀에 전달
