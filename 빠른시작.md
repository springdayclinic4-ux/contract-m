## 🎯 Vercel로 프론트엔드 배포 (1-Click)

### 1단계: GitHub에 코드 올리기

```bash
# Git 초기화
git init
git add .
git commit -m "Initial commit"

# GitHub 저장소 만들고 연결
git remote add origin https://github.com/your-username/theranova.git
git push -u origin main
```

### 2단계: Vercel 배포

1. https://vercel.com 접속 → GitHub 로그인
2. "Import Project" 클릭
3. GitHub 저장소 선택
4. **Root Directory: `frontend`** 설정
5. **Framework: Vite** 선택
6. Environment Variables 추가:
   ```
   VITE_API_URL=https://your-backend.onrender.com/api
   ```
7. "Deploy" 클릭!

완료! 2분 후 `https://your-app.vercel.app` 접속 가능

## 🚀 Render로 백엔드 배포

### 1단계: Neon에서 무료 DB 만들기

1. https://neon.tech 접속
2. "Sign Up" → GitHub 로그인
3. "Create Project" → 이름: `theranova-db`
4. Connection String 복사 (중요!)

### 2단계: Render 배포

1. https://render.com 접속
2. "New +" → "Web Service"
3. GitHub 저장소 연결
4. 설정:
   - Name: `theranova-backend`
   - Root Directory: `backend`
   - Build Command: `npm install && npx prisma generate`
   - Start Command: `npm start`
5. Environment Variables:
   ```
   NODE_ENV=production
   PORT=3001
   DATABASE_URL=[Neon에서 복사한 URL]
   JWT_SECRET=minimum-32-characters-random-string-here
   JWT_REFRESH_SECRET=another-32-characters-random-string
   MOCK_EMAIL=true
   ALLOWED_ORIGINS=*
   ```
6. "Create Web Service" 클릭

### 3단계: DB 마이그레이션

Render 대시보드 → Shell 탭:
```bash
npx prisma migrate deploy
```

완료! 3-5분 후 백엔드 준비 완료

## 🔗 연결하기

### Vercel 환경변수 업데이트
1. Vercel 프로젝트 → Settings → Environment Variables
2. `VITE_API_URL` 값을 Render URL로 변경:
   ```
   VITE_API_URL=https://theranova-backend.onrender.com/api
   ```
3. Redeploy

### Render CORS 업데이트
1. Render 프로젝트 → Environment
2. `ALLOWED_ORIGINS` 값을 Vercel URL로 변경:
   ```
   ALLOWED_ORIGINS=https://your-app.vercel.app
   ```
